Div e Classes

Div

Divs são elementos únicos ou seja usa um id para cada elemento

Divs são identificados na folha de estilos como #nomedoid

Sou um id
sou eu

Classes

Classes pode ser usados para vários elementos

Classes são identificados nas folhas de estilos como .nomedaclasse

Sou uma classe

Importante saber que que ID é mais forte que o Class, funcionando mesmo na hierarquia de cima para baixo.
