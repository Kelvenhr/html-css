Tags aprendidas

Strong

podemos deixar um texto escuro usando a tag <strong></strong> como agora.

Imagem

podemos colocar imagens com a tag img

Favicon

Podemos adicionar um favicon em lá em cima com link e favicon em head

Itálico

podemos deixar um texto em itálico usando a tag em

Sublinhado

Podemos sublinar um texto usando a tag ins

Links

podemos criar links (ligações)

Neffex

e usamos os parâmetros target para mostrar se queremos que abra uma nova pagina pro link ou se vamos usar a página atual para ir ao link
_blank para uma nova página e _self para usar a página atual e o parâmetro rel="" para demonstrar a relação do link se é external = externo ou next para próxima e prev para anterior e podemos usar o nofollow para links pagos no nosso site.

Audio

Para colocar audio no nosso site é simples usamos a tag audio e dentro criamos a tag source:src para demonstrar aonde se encontra o audio e o type dele

Parâmetros do audio

para mostrar os controles usamos o parâmetro controls dentro da tag audio

e podemos também definir se o download do audio será automatico ou não com o parâmentro em audio preload="" podendo usar os valores auto, none ou metadata.

Video

podemos também adicionar video no nosso site

no servidor

no servidor usamos a tag video, dentro da tag usamos source:src para mostrar aonde está o video e o tipo do video, podemos usar os parâmetros controls para mostrar os controles do video e o parâmetro preload="" com os valores auto, none e metadata

ou hindexado

pegar um video do youtube ou vimeo é muito simples basta clicar em compartilhar e escolher a opção incorporar e lá estará um codigo pronto que é so colocar no html.
