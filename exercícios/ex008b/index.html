Outras formatações

Código fonte / pré formatação

dentro da tag <code></code> todas as letras tem o mesmo espaçamento como você pode ver agora.

e em casos que você queira representar um código podemos utilizar a tag pre junto da tag code:

        
    n = int(input('Digite um número: '))
    if n < 14:
        print('você tem menos que 14 anos.')
    else:
        print('você tem 14 anos ou mais.')

como representado a tag pre faz o trabalho de escrever o código como digitamos
e a tag code para deixar o mesmo espaçamentos entre as letras.
